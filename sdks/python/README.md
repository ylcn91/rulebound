# rulebound

Preview-tier Python SDK for the Rulebound server API.

The stable core for Rulebound v0.1 is the CLI + engine + MCP + CI deterministic
gate (see the [Maturity tiers](../../README.md#maturity-tiers) table in the root
README). Native SDKs — including this one — sit at the Preview tier: the
TypeScript SDK is canonical, the others mirror it via a separate parity matrix,
and breaking changes are possible before SDK 1.0.0.
